Serviços de Monitorização Urbana

C# / Sockets TCP Sistemas Distribuídos MongoDB Multithreading

Sobre o Projeto

Desenvolvimento de um sistema distribuído para simulação de uma infraestrutura de monitorização ambiental urbana (sob o conceito One Health).

O sistema foi construído com base numa arquitetura de três entidades comunicantes via rede (Sensor → Gateway → Servidor) e focado em resolver problemas de alta concorrência e partilha de recursos.

A Minha Contribuição (Servidor Central)

  • Comunicação e Concorrência: Implementação do Servidor recorrendo a Sockets TCP em C# e gestão de ligações simultâneas de múltiplos Gateways usando Threads e Mutexes.
  • Persistência Resiliente: Armazenamento das medições ambientais em base de dados MongoDB. Implementação de um sistema de segurança (fallback) para ficheiros JSON locais caso a base de dados ficasse offline.
  • Processamento de Dados: Reconstrução de pacotes e armazenamento de streams de vídeo recebidos pela rede (em formato .mp4).

Tecnologias Utilizadas

  • Linguagens & Redes: C#, Sockets TCP, Protocolo Textual Customizado.
  • Persistência e Concorrência: MongoDB, JSON, Threads, Mutexes.
