Marketplace de Veículos Usados

ASP.NET Core MVC SQL Server EF Core

Sobre o Projeto

Aplicação web completa para a compra e venda de veículos usados, desenvolvida em equipa no âmbito académico.

A plataforma permite a interação entre compradores, vendedores e administradores, oferecendo funcionalidades como pesquisa avançada, reservas de veículos, comunicação entre utilizadores e gestão administrativa centralizada.

Funcionalidades

  • Perfis diferenciados para compradores, vendedores e administradores
  • Pesquisa avançada com filtros dinâmicos e ordenação
  • Sistema de reservas com controlo automático do estado dos anúncios
  • Comunicação direta entre compradores e vendedores
  • Área administrativa (backoffice) para gestão e moderação da plataforma
  • Avaliação de vendedores e sistema de notificações automáticas

Tecnologias Utilizadas

  • Backend: ASP.NET Core MVC
  • Base de Dados: SQL Server com Entity Framework Core
  • Frontend: HTML, CSS, JavaScript e Razor Views
  • Arquitetura: Organização por camadas, com separação entre lógica e dados

Desafios e Aprendizagens

Este projeto foi desenvolvido em equipa, permitindo consolidar competências de trabalho colaborativo, planeamento e gestão de código com Git.

O principal foco esteve na modelação de dados e na implementação de regras de negócio, garantindo a coerência dos estados dos anúncios, a integridade da informação e a correta separação de permissões entre os diferentes perfis de utilizador.
