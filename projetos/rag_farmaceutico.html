Assistente RAG Farmacêutico

LangChain Qdrant FastAPI RAG

Sobre o Projeto

Sistema inteligente baseado em Retrieval-Augmented Generation (RAG) desenvolvido para otimizar o acesso, pesquisa e análise de extensa documentação técnica e relatórios farmacêuticos.

O projeto focou-se em garantir respostas precisas a partir de uma base de conhecimento privada, mitigando "alucinações" dos Modelos de Linguagem (LLMs). A arquitetura do sistema foi cuidadosamente planeada com uma separação rigorosa entre os componentes de backend e frontend para maximizar a manutenção e escalabilidade em padrões de desenvolvimento profissional.

Funcionalidades

  • Pesquisa semântica em base de dados de documentos farmacêuticos.
  • Geração de respostas precisas baseadas exclusivamente no contexto recuperado (RAG).
  • Painel para visualização e gestão da documentação.
  • Registo de auditoria (audit trail) do sistema para rastreabilidade de queries.
  • Interface intuitiva que separa perfeitamente a lógica de negócio (backend) da apresentação (frontend).

Tecnologias Utilizadas

  • Orquestração e IA: LangChain, Claude 3.5 Sonnet (Anthropic), Gemini Embedding 2 (Google).
  • Backend e Base de Dados: Python, FastAPI, base de dados vetorial Qdrant (pesquisa híbrida).
  • Deployment e Compliance: Docker, conformidade com RGPD e EU AI Act (rastreabilidade local).
